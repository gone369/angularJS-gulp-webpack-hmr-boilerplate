# Installation
```bash
npm install
```

# Usage

## Serving For Development
```bash
gulp
```
open browser and go to http://localhost:1337

## Building For Distribution
```bash
gulp dist
cd build/
node server.js
```

## Editing Configs
```bash
cd config
vim config.js
```

## Editing eslint Configs
```bash
vim .eslintrc
```
